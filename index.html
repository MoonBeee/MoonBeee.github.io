Examples of Work

Zombie/Human Simulator

C++

For this console based application I was tasked to create a simulation. The 'H's represent humans and the 'Z's represent zombies. The humans move in a 4-directional pattern and replicate if they survive a predetermined number of turns. The zombies move in an 8-directional pattern and consume humans in their vicinty, after a predetermined number of turn they convert a human in an adjacent cell into a zombie.

This application was a focus on object oriented programming. Both human and zombies are a sub-class of 'organism'. Every turn the organisms "move" function was called as the application iterated over the 2-dimensional plane showcasing the power of polymorphism. As this was built in C++ this was also a create learning expirience of how pointers and the garabage collector work.

This was one of my favourite applications I have built to date. I learned a lot about pointers and C++ operator overloading as well as furthering my knowledge of OOP.
